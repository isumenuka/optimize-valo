## 📊 What Each Script Does

### `PLAY_VALORANT.bat` — Master Launcher
| # | Tweak | Effect |
|---|-------|--------|
| PRE | Kill all background apps | Free RAM + CPU for Valorant |
| 1-10 | Power plan, timers, registry | Lower DPC latency |
| 11-20 | CPU affinity, process priority | Valorant gets dedicated cores |
| 21-30 | GPU + display tweaks | Reduce render latency |
| 31-41 | Network: QoS DSCP 46, LSO off, RSC off, Flow Control off | ExitLag-level packet priority |

### `DEEP_BOOST.bat` — Advanced System Tweaks
| Tweak | Why |
|-------|-----|
| Kill 14 scheduled tasks | No CPU spikes during gaming |
| USB Selective Suspend OFF | No mouse/keyboard micro-freezes |
| PCIe ASPM OFF | GPU stays at full power state |
| Power Throttling OFF | Win11 won't throttle game threads |
| Radeon Anti-Lag ON | Driver-level, safe for Vanguard |
| AMD ULPS OFF | No stutter when loading maps |
| Prefetch/Superfetch OFF | NVMe doesn't need it |
| UDP socket buffers 64KB | Less packet loss under load |

### `GAMING_BOOST.bat` — Runtime Keeper (auto-runs)
- Re-applies Realtime priority every **60 seconds**
- Re-applies QoS DSCP 46 if Windows resets it
- Kills any background processes that respawn
- Flushes DNS + clears standby RAM each cycle
- Shows live Valorant server ping each cycle
- **Auto-closes when Valorant exits**

### `SETUP_TOOLS.bat` — CLI Tool Installer (once)
Downloads from **Microsoft Sysinternals** (official):
- `pskill.exe` — better process killer
- `pslist.exe` — process list from CMD
- `autorunsc.exe` — startup manager
- `RAMMap.exe` — RAM flush
- Compiles `EmptyStandbyList.exe` + `PingCheck.exe` natively
